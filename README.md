This repository contains the source code for my portfolio website. Visit the live site to see my projects, stats, and more.

### Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

### Deployment

The site is deployed on GitHub Pages and automatically updates on push to the main branch via GitHub Actions.
